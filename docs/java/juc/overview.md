# JUC 并发编程

## 📚 学习来源

本系列内容整理自黑马程序员的 JUC 并发编程课程：

👉 [黑马程序员深入学习Java并发编程，JUC并发编程全套教程](https://www.bilibili.com/video/BV16J411h7Rd)



## 📦 预备知识

在学习 JUC 之前，建议具备以下基础：
* 掌握java基本语法，本系列基于jdk8，最好对函数式编程、lambda有一定了解
* 接触过Java Web开发、Jdbc开发、Web服务器、分布式框架
* 使用slf4j记录日志，lombok简化java bean编写
  
## 🧭 学习目标

通过本系列学习，你将掌握：

- Java 并发编程核心思想
- 线程与线程安全问题本质
- 锁机制（synchronized / Lock）
- 原子类与 CAS 原理
- 并发容器设计
- 线程池原理与实践
- 常用并发工具类（CountDownLatch / Semaphore 等）
